# 🎭 IndexTTS 情绪向量内核对照表 (内核版本: 8-Dim Vector)

本表用于精准定义 IndexTTS Dynamic Emotion 节点的 8 维情感偏移参数。

---

### 📋 一、 维度功能定义 (UI 顺序与中文对照)

| 索引 | 界面标签 | 核心受控听感 | 调音逻辑 (数值正向增加) |
| :--- | :--- | :--- | :--- |
| **[0]** | **开心 (Happy)** | **明亮度** | 声音变清脆，嘴角上扬感增强，语速微快。 |
| **[1]** | **愤怒 (Angry)** | **压迫感** | 气息爆发力增强，喉音质感变重，攻击性强。 |
| **[2]** | **悲伤 (Sad)** | **厚重感** | 音色变闷、变厚，磁性/沙哑度增加，语调下沉。 |
| **[3]** | **恐惧 (Fear)** | **不稳定性** | 声线产生颤抖感，增加急促的吸气声，情绪紧迫。 |
| **[4]** | **厌恶 (Hate)** | **排斥感** | 增加鼻腔共鸣，语调冷淡、不屑、拒人千里。 |
| **[5]** | **低落 (Low)** | **能量值** | 声音变瘫软、松散，拖音增加，共鸣点靠后。 |
| **[6]** | **惊讶 (Surprise)** | **动态起伏** | 语调跨度变大，Pitch 波动剧烈，表现震惊/意外。 |
| **[7]** | **平静 (Neutral)** | **基准校准** | **强制保持 0.00**，作为模型情感归位的参考。 |

### 💡 三、 调音核心准则

1. **安全区 (0.10 - 0.60)**:
   在此区间内微调，声音最接近真人，适合绝大部分叙事和日常对话。

2. **极限区 (0.80 - 1.40)**:
   用于戏剧冲突顶点（如暴怒、狂喜）。注意：数值越高，模型越容易产生“电音感”或“撕裂感”。

3. **互斥补偿**:
   若觉得声音太“丧”，适当在 [0] 位补一点 Happy；若觉得声音太“飘”，适当在 [2] 位补一点 Sad。

4. **Neutral 强制位**:
   索引 [7] 必须永远为 0，否则可能导致模型声线整体偏移。
